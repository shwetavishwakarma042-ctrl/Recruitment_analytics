# Recruitment Analytics Dashboard (Excel)

## Project Overview
This project analyzes recruitment data to understand the hiring pipeline, identify candidate drop-off stages, and evaluate hiring efficiency. Using Microsoft Excel, I created an interactive analytics dashboard that visualizes key recruitment metrics and provides insights into the hiring process.

## Objectives
- Analyze candidate progression through different recruitment stages.
- Identify where the majority of candidate drop-offs occur.
- Evaluate the average time taken to hire candidates.
- Analyze the salary gap between candidate expectations and company offers.
- Build an interactive dashboard for recruitment performance monitoring.

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Data Visualization (Column Charts, Pie Charts, Histogram)
- Slicers for Interactive Filtering
- Dashboard Design

## Dashboard Features
The dashboard includes the following components:

- **KPI Metrics**
  - Total Candidates
  - Successfully Hired
  - Hiring Rate
  - Average Time to Hire

- **Recruitment Funnel Analysis**
  - Visualizes candidate progression through stages such as Screening, Interview, Offer, and Hiring.

- **Hiring Conversion Chart**
  - Shows the percentage of candidates successfully hired.

- **Time-to-Hire Analysis**
  - Displays the average number of days required to hire a candidate.

- **Salary Gap Analysis**
  - Compares candidate salary expectations with offered salaries.

- **Histogram Distribution**
  - Shows the distribution of time-to-hire across different ranges.

## Key Insights
- A significant number of candidates drop off during the **interview stage**.
- The **average hiring cycle is approximately 21 days**.
- Salary gaps between candidate expectations and company offers may influence hiring decisions.
- The hiring conversion rate indicates opportunities for improving the recruitment pipeline.
